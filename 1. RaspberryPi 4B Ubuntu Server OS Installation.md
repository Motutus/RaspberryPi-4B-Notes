# 树莓派4B Ubuntu Server 系统安装

	文档创建时间：2023年02月06日

<br><br>

### 安装准备

	1. MicroSD卡（至少 8G 内存）
	2. SD卡读卡器
	3. Raspberry Pi 4B
	4. PC + 网络环境
